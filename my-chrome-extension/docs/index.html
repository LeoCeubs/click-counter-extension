Bootcamp Helper

Extensão para Chrome (Manifest V3) — projeto do Bootcamp II.

Funcionalidade

Esta extensão simples adiciona um botão no seu navegador que, quando clicado, realça todos os links (<a>) da página ativa com uma borda colorida.

Instalação manual (modo desenvolvedor)

  1. Acesse chrome://extensions no seu navegador Chrome.
  2. Ative o "Modo do desenvolvedor" (Developer mode) no canto superior direito.
  3. Clique em "Carregar sem compactar" (Load unpacked) e selecione a pasta raiz do projeto.

Permissões

A extensão requer as seguintes permissões para funcionar:

  • storage: Para armazenar dados localmente no navegador (como a data de instalação).
  • activeTab: Para injetar o script na guia ativa quando o usuário clica no botão.
